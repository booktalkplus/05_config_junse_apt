# 전세낀 - 개인정보 처리 지침

**최종 수정일**: 2026-02-23  
**적용 대상**: 전세낀 - 전세낀 아파트 찾기 (Chrome 확장 프로그램)

---

## 1. 개요

전세낀('이하 '본 서비스')은 네이버 부동산에 등록된 아파트 매물 중 전세낀(임대차 승계) 매물을 찾아 리스트로 제공하는 Chrome 확장 프로그램입니다. 본 지침은 본 서비스가 수집·저장·이용하는 정보에 대해 설명합니다.

---

## 2. 수집·저장하는 정보

### 2-1. 로컬 저장 정보 (개발자 서버로 전송되지 않음)

| 정보 유형 | 저장 위치 | 용도 | 보유 기간 |
|-----------|-----------|------|-----------|
| 사용자 설정 | chrome.storage.local | 보증금 필터, 키워드 필터, 알림 ON/OFF, 사용자 정의 키워드 목록 | 확장 프로그램 설치 기간 |
| 마지막 검색 매물 ID | chrome.storage.local | 사용자 편의(이전 검색 재실행) | 확장 프로그램 설치 기간 |
| 분석 결과 | chrome.storage.local | 최근 1건의 매물 분석 결과만 보관(리포트 표시용) | 새 분석 시 자동 폐기 |
| 공지/설정 캐시 | chrome.storage.local | 원격 공지사항·설정 JSON 로컬 캐시 | 최신 조회 시 갱신 |
| 약속 이행 횟수 | localStorage | 리포트 내 '약속' 체크박스 UI 상태 | 확장 프로그램 설치 기간 |

### 2-2. 수집하지 않는 정보

본 서비스는 다음 정보를 **수집·저장·전송하지 않습니다**.

- 성명, 이메일, 전화번호, 주소 등 개인 식별 정보
- Google 계정·Chrome 프로필과 연계되는 고유 식별자
- 사용자의 네이버 계정 로그인 정보(비밀번호 등)

---

## 3. 외부 통신

### 3-1. 수신만 하는 통신 (GET 요청)

본 서비스는 **설정·공지용 JSON 파일**만 다음 주소에서 가져옵니다.  
이 과정에서 **사용자 데이터는 전송되지 않습니다**.

- 공지사항 (notice.json)
- 원격 설정 (config.json)
- 후원 배너 설정 (sponsor.json)
- API 엔드포인트 설정 (api.json)

> 위 파일들은 GitHub Raw URL에서 제공되며, 사용자 식별 정보를 포함한 요청 파라미터는 사용하지 않습니다.

### 3-2. 네이버 부동산 접근

- 본 서비스는 **사용자의 브라우저**에서 네이버 부동산 페이지에 접근하여 매물 데이터를 분석합니다.
- 이 과정은 사용자가 직접 네이버 부동산을 방문하는 것과 동일한 방식으로 이루어지며, **수집된 매물 데이터는 사용자 기기 내에서만 처리**됩니다.
- 네이버 로그인 세션 확인용으로 해당 도메인의 쿠키 존재 여부만 로컬에서 확인하며, 쿠키 값 자체는 외부로 전송되지 않습니다.

---

## 4. 권한별 사용 목적

| 권한 | 사용 목적 |
|------|------------|
| storage | 사용자 설정·분석 결과를 로컬에 저장 |
| tabs | 현재 열린 네이버 부동산 탭 감지 및 분석 탭 관리 |
| scripting | 네이버 부동산 페이지에 분석 스크립트 주입 |
| activeTab | 사용자가 클릭한 탭의 URL/상태 확인 |
| cookies | 네이버 로그인 세션 여부 확인(로컬에서만 사용) |
| alarms | 스케줄 기반 작업 실행 |
| debugger | 네이버 부동산 API 응답을 가로채어 매물 데이터 수집(감청 방식) |
| notifications | 수집 완료 시 브라우저 알림 표시 |

---

## 5. 데이터 보안

- 수집된 API 원본은 분석 후 **즉시 파기**하며, 리포트 표시에 필요한 최소 필드만 정제된 형태로 보관합니다.
- 분석 결과는 **가장 최근 1건만** 보관하며, 새 분석 시 이전 데이터는 자동으로 삭제됩니다.
- 사용자 데이터는 **개발자 서버로 전송되지 않으며**, 모든 처리와 저장은 사용자의 Chrome 브라우저 내에서만 이루어집니다.

---

## 6. 사용자 권리

- **확장 프로그램 제거** 시 `chrome.storage.local` 및 `localStorage`에 저장된 데이터는 브라우저 정책에 따라 삭제됩니다.
- 설정 초기화가 필요한 경우, Chrome 확장 프로그램 관리 페이지에서 해당 확장 프로그램을 제거 후 재설치하면 됩니다.
