# 🌐 원격 설정 시스템 (Remote Config)

이 폴더는 '전세낀' 크롬 확장 프로그램의 동작을 앱 배포 없이 실시간으로 제어하기 위한 설정 파일들을 관리합니다.

## 📂 파일 구성 및 역할

1. **[config.json](./config.json)**: 시스템 코어 설정
    - 수집 매물 제한(`ENRICHMENT_LIMIT`)
    - 수집 간 랜덤 지연 시간(`ENRICHMENT_DELAY_RANGE`)
    - 오픈톡방 링크 및 참여 코드(`KAKAO_TALK_LINK`, `KAKAO_TALK_CODE`)
    - 분석 가이트 문구 등

2. **[messages.json](./messages.json)**: 로딩 화면 메시지
    - 로딩 화면 하단에 롤링되는 후원 유도 및 개발자 메시지 모음

3. **[notice.json](./notice.json)**: 전역 공지사항
    - 팝업 상단 및 리포트 상단에 표시될 긴급 공지 및 알림 메시지

## 🛡️ 중요 보안 원칙

- **민감 정보 포함 금지**: 쿠팡 파트너스 링크, API 키 등 보안이 중요한 정보는 절대로 이 폴더의 파일(원격 파일)에 기록하지 않습니다.
- **로컬 우선 보호**: 앱의 `config-manager.js`는 원격 설정을 가져올 때 `UTILITY` 항목(수익 링크 등)을 자동으로 제외하도록 설계되어 외부 주입으로부터 보호됩니다.
- **민감 정보 수정 방법**: 수익 링크 등은 반드시 확장 프로그램 소스 내의 `chrome-extension/config.js`에서 직접 수정하여 배포해야 합니다.

## 🚀 반영 방법 (실시간 제어)

1. 이 폴더의 내용을 수정 후 깃허브에 푸시합니다.
2. 깃허브 웹사이트에서 해당 파일의 **[Raw]** 버튼을 클릭하여 URL을 복사합니다.
3. 확장 프로그램의 `config.js` 내 `REMOTE_URLS` 항목에 해당 주소를 입력합니다.
4. 이후부터는 사용자가 앱을 켜는 순간 깃허브의 최신 내용이 즉시 반영됩니다.
